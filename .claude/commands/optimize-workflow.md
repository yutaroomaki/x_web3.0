Use the workflow-coordinator subagent to optimize the development workflow.

Current workflow description: $ARGUMENTS

The subagent should:
1. Analyze the current development process
2. Identify and classify all tasks
3. Build dependency graph (DAG)
4. Identify tasks that can run in parallel
5. Calculate critical path
6. Propose optimized workflow with time savings

Output a workflow optimization report with DAG diagrams and improvement proposals.
