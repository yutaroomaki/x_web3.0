Use the database-designer subagent to design the database schema.

Requirements: $ARGUMENTS

The subagent should:
1. Extract entities from business requirements
2. Create ER diagram
3. Apply normalization (to appropriate level)
4. Define primary keys, foreign keys, and constraints
5. Design indexing strategy for performance
6. Generate migration scripts

Output a complete database design specification with ER diagrams and SQL scripts.
