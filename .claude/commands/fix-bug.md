---

Use the bug-fixer subagent to analyze and fix the bug.

Bug details: $ARGUMENTS

The subagent should:
1. Analyze error logs and stack traces
2. Identify reproduction steps
3. Search related code
4. Diagnose root cause
5. Implement the fix
6. Add regression tests
7. Document the fix and prevention measures

Output a complete bug fix with tests and documentation.
