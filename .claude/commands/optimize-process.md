Use the process-optimizer subagent to analyze and optimize development processes.

Process scope: $ARGUMENTS

The subagent should:
1. Visualize current process (Value Stream Mapping)
2. Collect metrics (Lead Time, Cycle Time, Throughput)
3. Analyze bottlenecks (5 Whys, Theory of Constraints)
4. Generate improvement proposals (Quick wins, Long-term)
5. Identify automation opportunities
6. Calculate ROI (Time saved vs Implementation cost)
7. Create experiment plans (A/B testing, PDCA)
8. Build continuous improvement loop

Output complete process optimization suite with:
- Value Stream Map (Process steps, wait times, bottlenecks)
- Bottleneck analysis (TypeScript implementation with TOC)
- Automation opportunity detection (ROI calculator)
- Developer Experience metrics (Setup time, Build time, Tool satisfaction)
- Improvement recommendations (Prioritized by ROI)
- PDCA experiment templates
- Productivity dashboard
