Use the learning-agent subagent to extract and document organizational knowledge.

Learning scope: $ARGUMENTS

The subagent should:
1. Extract lessons from incidents/projects
2. Identify patterns and anti-patterns
3. Create best practices documentation
4. Update knowledge base
5. Plan team sharing sessions
6. Create mentoring matches
7. Generate learning roadmaps
8. Analyze knowledge gaps

Output complete organizational learning suite with:
- Lessons Learned templates (structured extraction)
- Best Practices catalog (50+ items with code examples)
- Anti-Patterns registry (common mistakes to avoid)
- Knowledge Base builder (automated extraction from postmortems)
- Learning paths (onboarding roadmaps by role)
- Team sharing session plans
- Knowledge gap analysis
